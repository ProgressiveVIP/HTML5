What is HTML?

HTML is the standard markup language for creating Web pages.

Topic

Theory

Examples

All HTML documents must start with a document type declaration: < !DOCTYPE html >
The HTML document itself begins with
and ends with 		< html >
< /html >
The visible part of the HTML document is between < body > and < / body >
The HTML < head > element is a container for metadata (data about the
HTML document: document title, character set, styles,
links, scripts, and other meta information). 		< head > . . . < / head >
Headings are defined with the < h1 > to < h6 > tags.
Attributes provide additional information about an element language, charset, author, keywords and etc.
